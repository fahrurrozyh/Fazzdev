BEKERJA BERSAMA GIT

git config
mengatur akun, nama, dan email untuk git
git init
menjadikan "folder biasa" menjadi "repository git"
git add
menyimpan perubahan dari "working tree" ke "staging area"
git commit
menyimpan perubahan dari "staging area" ke "history / log"
git status
memunculkan status lengkap kejadian di "repository git"
git log
melihat history perubahan yang sudah di commit sebelumnya
git checkout
mengembalikan perubahan yang sudah di commit sebelumnya
git log
melihat history perubahan yang sudah di commit sebelumnya
