GIT BRANCH & MERGE

git branch
menampilkan branch yang ada
git branch nama_cabang
menambahan branch "nama_cabang"
git log --all --decorate --oneline --graph
menampilkan history graph seperti di github
alias graph="git log --all --decorate --oneline --graph"
membuat alias biar lebih simple nggak ngetik panjang2
git checkout nama_cabang
pindah ke branch lain "nama_cabang"
git merge nama_cabang
menyatukan dua cabang menjadi satu saja
git branch --merged
melihat branch yang sudah dimerge
git branch -d nama_cabang (pakai d kecil)
menghapus brach (muncul pesan jika belum di merge)
git branch -D nama_cabang (pakai D kecil)
menghapus brach (langsung terhapus meskipun belum di merge)
