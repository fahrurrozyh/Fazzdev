DISPLAY

  • Tag DIV dan SPAN adalah Tag Pengelompokan
  • semua TAG ada dalam KOTAK, DISPLAY mengatur KOTAK
  • DIV punya display BLOCK, SPAN punya display INLINE
  • DISPLAY ada 4: inline, inline-block, block, none
  • inline: tidak buat baris baru, tinggi lebar sesuai konten
  • inline: dimensi tak bisa diubah, margin padding hanya horizontal
  • inline block: tidak ada tag secara default pakai display ini
  • inline-block: inline yang bisa diatur dimensi (tinggi lebar)nya
  • block: buat baris baru, lebar memenuhi browser & parent
  • block: bisa diatur tinggi dan lebarnya, contoh: H1, H2, P, dst
